Пошук фільмів.<br>
Напиши застосунок пошуку фільмів за ключовим словом.<br>
Сервіс пошуку фільмів TMDB.<br>
Для сповіщень використовуйте бібліотеку React Hot Toast.<br>
Під час натискання на зображення галереї повинно відкриватися модальне вікно, <br>
яке відображатиме додаткову інформацію про фільм у великому форматі.<br>
Доповни свій застосунок пошуку фільмів пагінацією.<br>
Для відображення елементів пагінації використовуйте біліотеку React Paginate.<br>
Керування станом запитів використовуйте бібліотеку TanStack Query.<br>
